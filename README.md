---

## ⚡ Quick Start

### 1. Start Development Server
```bash
npm run dev
```
Visit `http://localhost:3000`

### 2. Access Content Management
Go to `http://localhost:3000/studio` to manage neighborhoods, properties, and blog posts.

### 3. Build for Production
```bash
npm run build
npm start
```
