## How to Use

### Method 1: FASTEST (30 seconds)

Run the automated seed script:

```bash
# 1. Get Sanity API token from sanity.io dashboard
# 2. Run script
SANITY_API_TOKEN=your_token node scripts/seed-real-data.js

# Wait 30 seconds for:
# ✅ 8 neighborhoods created
# ✅ 13 sold properties created
# ✅ 5 blog posts created
```

Then:
1. Go to http://localhost:3000/studio
2. Add hero images to each neighborhood
3. Verify content appears on site
4. Deploy to Vercel

### Method 2: SAFEST (2-3 hours)

Manually populate through Sanity Studio:

```bash
# 1. Start dev server
npm run dev

# 2. Open Sanity Studio
# Go to http://localhost:3000/studio

# 3. Use POPULATE_NEIGHBORHOODS.md for each neighborhood
# Create 8 neighborhoods with all details

# 4. Add images

# 5. Deploy
```

## Common Questions

**Q: Which method should I choose?**  
A: If you want it live today → use automated script. If you want to customize → use manual.

**Q: Can I change the content later?**  
A: Yes! Go to `/studio` anytime to edit neighborhoods, add images, update market stats.

**Q: Do I need to update the seed script after populating?**  
A: No, the script is just for initial population. Edit content in Sanity Studio afterward.

**Q: What if I only want 5 neighborhoods?**  
A: Just populate 5 via manual method. Or run script and delete the ones you don't want.

**Q: Can I add more neighborhoods later?**  
A: Yes! Go to `/studio`, click "Create", add new neighborhood.
